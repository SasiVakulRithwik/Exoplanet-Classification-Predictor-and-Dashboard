🚀 Project Overview

The Exoplanet Classification Dashboard is a comprehensive machine learning web application that classifies celestial objects as exoplanets, candidates, or false positives using NASA's Kepler mission data. The system provides real-time predictions, comprehensive analytics, and interactive visualizations to help astronomers and researchers identify potential exoplanets more efficiently.

🎯 Project Goals

  • Automate the classification of celestial objects using machine learning
  • Provide an intuitive interface for astronomers and researchers
  • Offer comprehensive analytics and visualization tools
  • Enable real-time predictions with confidence scores
  • Support data-driven decision making in exoplanet research

Feature Categories

🌌 Principal Components
  • PC1, PC2, PC3 - Dimensionality reduced features from original light curve data
📍 Celestial Coordinates
  • Right Ascension (ra) - Celestial longitude coordinate
  • Declination (dec) - Celestial latitude coordinate
⭐ Stellar Properties
  • Star Temperature - Effective temperature in Kelvin
  • Surface Gravity (logg) - Logarithm of surface gravity
  • Star Radius - Radius in solar units
  • Star Mass - Mass in solar units
🪐 Planetary Characteristics
  • Planet Radius - Radius in Earth radii units
  • Orbital Period - Orbital period in days
📊 Photometric Data
  • J, H, K Magnitudes - Infrared brightness measurements
  • Kepler Magnitude - Visible light brightness
📈 Light Curve Features
  • Light Curve Time 0 - Reference time for light curve analysis

Target Classification

❌
False Positive (0)

Signals caused by instrumental errors, stellar activity, or binary star systems that mimic planetary transits

  • Instrumental artifacts
  • Stellar variability
  • Eclipsing binaries
  • Systematic errors
🔍
Candidate (1)

Potential exoplanets requiring further verification through additional observations and analysis

  • Promising transit signals
  • Requires confirmation
  • Follow-up observations needed
  • Statistical validation required
✅
Confirmed Planet (2)

Verified exoplanets with multiple detection confirmations through different methods

  • Multiple detection methods
  • Statistical significance
  • Peer-reviewed confirmation
  • Published in scientific literature
