📁 Dataset Information

Learn about the Exoplanet dataset and its features

Dataset Overview

The dataset contains astronomical observations from NASA's Kepler mission with features derived from light curve analysis and stellar properties.

Key Features:

  • PC1, PC2, PC3: Principal Components from dimensionality reduction
  • ra, dec: Celestial coordinates (Right Ascension, Declination)
  • Magnitudes (J, H, K, Kepler): Brightness measurements in different filters
  • Planet Properties: Radius, Orbital Period
  • Star Properties: Temperature, Surface Gravity, Radius, Mass
  • Light Curve Data: Time series photometric measurements

Target Classes:

False Positive (0)
Candidate (1)
Confirmed Planet (2)

📚 Project Documentation

Technical details and implementation information

Project Architecture

This application uses a machine learning model to classify celestial objects as exoplanets, candidates, or false positives based on astronomical data.

Technology Stack:

  • Backend: FastAPI (Python)
  • Frontend: HTML, CSS, JavaScript
  • ML Framework: Scikit-learn
  • Visualization: Chart.js
  • Data Processing: Pandas, NumPy

Model Details:

The classification model is trained on NASA's exoplanet dataset using supervised learning algorithms. It processes 16 input features to predict one of three classes.

API Endpoints:

GET /api/summary Returns dataset statistics
POST /api/predict Makes classification predictions
